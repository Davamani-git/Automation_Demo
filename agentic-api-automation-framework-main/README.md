# Agentic API Automation Framework

Enterprise API Automation Framework built using:

- Playwright
- JavaScript
- Node.js
- Layered Architecture

## Features

- REST API Testing
- Token Authentication
- Service Layer
- Client Layer
- Validators
- Payload Builders
- Logging
- Environment Support
- HTML Reports

## Project Structure

```
tests
src
   api
   services
   clients
   payloads
   validators
config
utils
```

## Run Tests

```bash
npm install
npx playwright test
```
